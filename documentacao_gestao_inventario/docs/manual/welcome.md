# Página Inicial

## Visão Geral

A página inicial apresenta um dashboard com as principais informações do sistema:

- Resumo de Inventário
- Últimos Incidentes
- Indicadores de Status

## Menu Principal

O menu principal contém as seguintes opções:

- Inventário
  - Gestão de Inventário
  - Fluxo de Inventário
- Incidentes
- Configurações
